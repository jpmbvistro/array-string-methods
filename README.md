# JS String and Array Methods for Interviewing

## String Methods

### charAt

Returns the character at a specified zero-based index

#### Use:
```
<string>.charAt(<index>)
```


#### Example:
```
let stringy = Bubleh!
console.log( stringy.charAt(2) )
//Expect to print 'b'

console.log( stringy.charAt(6) )
//Expect to print '!'
```
